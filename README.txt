=======================================
   ФИНАНСОВЫЙ ТРЕКЕР
   Personal Finance Tracker
=======================================

Version: 2.0

ОПИСАНИЕ
--------
Веб-приложение для учёта личных финансов. Позволяет вести учёт доходов
и расходов, планировать бюджет, отслеживать регулярные платежи и
анализировать финансовое состояние.

ОСНОВНЫЕ ВОЗМОЖНОСТИ
--------------------
- Дашборд: статистика за период, баланс, остаток на начало периода,
  прогноз расходов, предстоящие платежи, светофор кассового разрыва.
- Операции: добавление, редактирование, удаление доходов и расходов.
  Фильтрация по периоду, типу, категории, дате. Пагинация.
- Регулярные платежи: настройка ежемесячных, еженедельных и ежегодных
  платежей. Автоматическое применение в указанные даты.
- Категории: иерархические категории (основная + подкатегории).
  Inline-редактирование прямо в таблице.
- Бюджеты: установка лимитов по категориям на месяц. Общая сводка
  с прогресс-баром.
- Аналитика: график доходов/расходов по месяцам (line chart),
  круговая диаграмма расходов по категориям, детализация с бюджетом.
- Планирование бюджета: расчёт аванса, остатка зарплаты, сумм
  для откладывания на регулярные платежи.
- Экспорт операций в CSV.
- Автоматическое резервное копирование базы данных (2 раза в день,
  хранение 30 последних копий).
- Авторизация: username + password (пароль хешируется через werkzeug).
- Docker: готовый Dockerfile и docker-compose.yml для запуска
  в контейнере.

СИСТЕМНЫЕ ТРЕБОВАНИЯ
--------------------
- Python 3.10+
- Flask 3.x
- SQLite 3
- APScheduler 3.x
- Werkzeug
- AdminLTE v4 (Bootstrap 5) — подключается через CDN
- Chart.js 4.4 — подключается через CDN

УСТАНОВКА И ЗАПУСК
------------------

1. Клонировать репозиторий:
   git clone <url-репозитория>
   cd finance_tracker

2. Создать виртуальное окружение и установить зависимости:
   python -m venv venv
   source venv/bin/activate   # Linux/Mac
   venv\Scripts\activate      # Windows
   pip install -r requirements.txt

3. Запустить приложение:
   flask run
   или
   python app.py

4. Открыть в браузере: http://127.0.0.1:5000

5. Войти с логином: admin, пароль: 12345
   (пароль можно изменить через переменную окружения APP_PASSWORD)

ЗАПУСК ЧЕРЕЗ DOCKER
-------------------
   docker compose up -d
   Приложение будет доступно на http://localhost:5000

ПЕРЕМЕННЫЕ ОКРУЖЕНИЯ
--------------------
SECRET_KEY     — секретный ключ Flask (по умолчанию: your-secret-key-here)
DB_PATH        — путь к файлу базы данных (по умолчанию: finance.db)
APP_PASSWORD   — пароль admin при первой инициализации (по умолчанию: 12345)
SALARY_DAY     — день зарплаты (по умолчанию: 10)
ADVANCE_DAY    — день аванса (по умолчанию: 25)

ПЕРИОДЫ УЧЁТА
-------------
Система использует два периода в месяце:
  10-24 — с 10 по 24 число (первая половина)
  25-09 — с 25 по 9 число следующего месяца (вторая половина)
Каждая операция автоматически привязывается к периоду по дате.

АВТОМАТИЧЕСКОЕ РЕЗЕРВНОЕ КОПИРОВАНИЕ
-------------------------------------
Резервные копии создаются в папке backups/ автоматически:
- при каждом запуске приложения
- по расписанию в 6:00 и 18:00 ежедневно
Хранятся 30 последних копий.
